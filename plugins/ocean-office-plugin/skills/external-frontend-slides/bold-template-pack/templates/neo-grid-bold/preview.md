## 视觉概览

一套厚重的编辑海报系统，建立在严格的 12 列 x 8 行块状网格之上，配灰黄素色上的霓虹黄色点缀。Space Grotesk 字重 700 严格大写承载每个展示时刻；JetBrains Mono 承载每个标签和元数据标签。每张幻灯片读起来像是被分割为彩色面板的杂志展开——纸张素色、墨黑色和电光柠檬黄在单元格间轮换角色。美学借鉴自当代编辑印刷、粗野主义年度报告和设计周展示的大众化海报端。

Neo-Grid Bold 是一套厚重的编辑海报系统，建立在单一结构理念之上：每张幻灯片是一个 12 列 x 8 行的 CSS 网格，从幻灯片边缘内缩 40px，单元格之间有 12px 间隙。构图通过将彩色面板分配给网格跨度来完成——grid-column: 4 / span 5 和 grid-row: 1 / span 5 就是描述布局的方式。网格是刚性的；视觉多样性来自 {colors.paper}、{colors.ink} 和 {colors.accent-lemon} 面板如何跨单元格排列。

## 预览要素

- 调色板：paper #F5F4EF；bg #ECECE8；ink #0A0A0A；accent-lemon #E6FF3D；muted #8A8A85；stage-bg #1A1A1A
- 字体排版：Space Grotesk；JetBrains Mono
- 标志性手法：通用 12 列 x 8 行 CSS 网格（{components.frame}）从每张幻灯片边缘内缩 40px，单元格之间有 12px 间隙。
- 标志性手法：三色面板系统：纸张素色（{colors.paper}）作为默认，墨黑色（{colors.ink}）用于反转块，点缀柠檬黄（{colors.accent-lemon}）用于信号块。
- 标志性手法：Space Grotesk 字重 700 大写配负字间距用于每个展示元素。
- 标志性手法：JetBrains Mono 大写配 0.08-0.12em 字间距用于每个标签、页码、轴标签。
- 标志性手法：标志性的角标和块标——小型 2x2 块状戳记作为装饰性身份标签。
