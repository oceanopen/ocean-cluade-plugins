# /ocean-office:external-html-ppt-skill

## 何时使用此命令

当用户需求属于以下场景时，选择此命令：

- **正式演示**：商务演讲、投资路演、产品发布、季度汇报
- **技术分享**：团队技术分享、工程 review、架构讲解
- **结构化文档**：周报、月报、项目总结、课程教案
- **演讲辅助**：需要演讲者模式、逐字稿、提词器的场景
- **小红书图文**：小红书风格的内容卡片
- **主题丰富**：需要从 36 种主题中匹配特定风格

## 实施步骤

1. **理解需求**：询问演示文稿的用途、受众、页数和风格偏好
2. **委派执行**：调用 `external-html-ppt-skill` 技能完成所有制作工作
3. **交付文件**：输出可直接在浏览器打开的 HTML 文件
