## 视觉概览

一套 Windows 95 / 98 桌面操作系统美学渲染为演示系统。每张幻灯片都是一个窗口——斜面边框、海军蓝渐变标题栏、MS Sans Serif 正文字体，图表区域、分组框和面板排列得如同 1995 年的软件 UI。调色板使用原始 Win9x 系统颜色（灰色按钮面、海军蓝标题栏、白色凹陷输入框）配复古点缀色（DOS 绿、砖红、芥末黄、青蓝）保留用于状态文字和图表数据。像素字体（Press Start 2P）和终端字体（VT323）少量出现作为怀旧点缀。整体效果一半是活泼的怀旧，一半是功能性仪表板——读起来像是运行在 CRT 显示器上的软件产品。

Retro Windows 是一套 Windows 95 / 98 桌面操作系统美学渲染为幻灯片模板。每张幻灯片结构化为一个窗口——带有海军蓝渐变标题栏的斜面矩形边框、右上角的三个按钮图标（_、[]、X）和包含应用程序风格内容的主体区域。构图是"这张幻灯片是运行在 1995 年桌面上的软件，内容是软件所显示的"。理念是完整的：没有现代演示意义上的幻灯片标题，只有样式化为文件名的窗口标题（README.DOC、DATAVIEW.CSV、METRICS.LOG）。

## 预览要素

- 调色板：bg-gray #C0C0C0；bg-light #D4D0C8；bg-dark #808080；white #FFFFFF；black #000000；text-dark #222222；blue-navy #000080；blue-bright #0000A0
- 字体排版：MS Sans Serif
- 标志性手法：每张幻灯片是一个 {components.win-window}——带海军蓝渐变标题栏和三个系统按钮（_、[]、X）的斜面边框。
- 标志性手法：固定的 3px 周期 CRT 扫描线叠加层（{components.crt-overlay}）以 3% 透明度位于所有内容之上。
- 标志性手法：斜面式深度：凸起（{components.panel-raised}、{components.btn-retro}）和凹陷（{components.panel-sunken}、{components.group-box}）——无模糊阴影。
- 标志性手法：字体栈是 MS Sans Serif / Segoe UI / Tahoma 备选，配 Press Start 2P 和 VT323 作为怀旧点缀。
- 标志性手法：状态颜色（绿色/红色/黄色/青色）具有语义含义：绿色 = 正常，红色 = 警告，黄色 = 中等，青色 = 第三方数据。
