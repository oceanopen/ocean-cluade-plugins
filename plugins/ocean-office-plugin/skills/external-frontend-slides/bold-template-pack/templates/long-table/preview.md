## 视觉概览

一套温暖的、单墨色编辑系统，处于晚餐俱乐部海报、小批量杂志或 Risograph 印刷节目的层次。整个系统以一种墨色运行——温暖的铁锈陶土色（#B53D2A）——在黄油奶油色纸面地面（#FAF1E2）上，配以微妙的 4px 径向点纹理叠加层赋予表面"印刷纸张"质感。展示字体使用 Bricolage Grotesque 字重 700-800 大写；正文和元数据使用 Fraunces 衬线体字重 400-600，启用光学尺寸轴。药丸按钮、轮廓版本徽章、斜体版本数字和虚线/实线 1.5px 边框完成了印刷节目词汇。

Long Table 是一套单墨色编辑系统，处于晚餐俱乐部海报、Risograph 杂志或小型出版社晚宴节目的层次。基础理念是单色的：系统中每个可见标记——每个标题、每个段落、每个边框、每条线、每个药丸、每个页码——都以单一温暖铁锈陶土色墨水（{colors.ink}——#B53D2A）在黄油奶油色纸面地面（{colors.paper}——#FAF1E2）上渲染。唯一的色彩变化是透明度：同一墨水在 100% 用于主要标记，78% 用于弱化元数据，32% 用于虚线内部分隔线，10% 用于纸张纹理点。

## 预览要素

- 调色板：paper #FAF1E2；paper-d #F2E5CF；paper-vd #E8D7B6；ink #B53D2A；ink-dp #8E2D1F；rule #B53D2A
- 字体排版：选择后参见完整设计文档。
- 标志性手法：Bricolage Grotesque 字重 700 和 800 严格大写承载每个展示时刻——封面、标题、卡片标题、课程名称、引用正文、信息值、人物标签。Bricolage 是一种宽阔、略窄的无衬线体，个性强烈；其大写...
- 标志性手法：Fraunces 字重 400-600 承载每个正文段落、每个元数据字段、每个药丸、每个页码、每个版本标签。斜体 Fraunces 是默认正文风格——倾斜的衬线体是系统的正文声音，赋予温暖和编辑个性...
- 标志性手法：一种墨色（{colors.ink}——温暖铁锈陶土色）在奶油色纸张（{colors.paper}）上。透明度变体是唯一的颜色变化。
- 标志性手法：Bricolage Grotesque 字重 700-800 大写用于每个展示时刻；Fraunces 400-600 斜体默认用于每个正文和元数据时刻。
- 标志性手法：微妙的 4px 径向点纸张纹理通过 .stage::before 应用于每张幻灯片——远距离不可见，近距离可感知。
