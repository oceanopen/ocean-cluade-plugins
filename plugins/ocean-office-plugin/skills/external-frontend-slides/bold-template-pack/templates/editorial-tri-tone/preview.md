## 视觉概览

一套文学杂志遇上年度报告的演示系统，建立在严格的三色调色板之上——腮红粉、金黄黄油和深酒红。尽管有十一个 CSS 变量名，整个系统中只存在三个十六进制值；每个别名都是其中之一的语义重命名。标题使用 Bricolage Grotesque（一种具有光学尺寸轴的可变无衬线体），极端字重和负字间距。Instrument Serif（仅斜体裁切）作为表现性点缀字体出现，用于章节数字、引用文字、年份和签名。JetBrains Mono 以紧凑大写间距承载所有元数据、标签和章节标记。美学是"独立艺术出版物"——那种有版权页、手工编号版本和编辑台的风格。

Editorial Tri-Tone 是一套文学杂志演示系统，建立在可能最严格的调色板之上：三个十六进制值，十一个 CSS 变量名。命名系统揭示了编辑意图——--pink 和 --sky 指向同一腮红色；--cream、--butter、--lime 和 --terracotta 都解析为同一金黄色；--burgundy、--navy、--forest 和 --ink 全部坍塌为同一深酒红色。别名的存在是为了传达颜色在上下文中的角色，而非引入变化。

## 预览要素

- 调色板：pink #F2B6C6；butter #F2D86A；burgundy #7A1F35
- 字体排版：Bricolage Grotesque；Instrument Serif；JetBrains Mono
- 标志性手法：三色调色板配语义别名：腮红粉（#F2B6C6）、金黄黄油（#F2D86A）、深酒红（#7A1F35）。
- 标志性手法：混合字体：Bricolage Grotesque 无衬线体用于展示和正文；Instrument Serif 用于表现性点缀数字、引用和年份；JetBrains Mono 用于所有标签。
- 标志性手法：Bricolage Grotesque 标题中的 em 标签始终触发切换到 Instrument Serif 斜体——系统的主要排版混搭。
- 标志性手法：药丸形（border-radius 999px）是通用标签组件；在封面、图表图例和结束页三种不同尺寸使用。
- 标志性手法：数值卡片拥有宽松的 28px border-radius——柔软、圆润、非粗野主义。
