## 视觉概览

一套 WPA 海报遇上政治竞选的演示系统，建立在三种字体和五种颜色（精简为三种功能角色）的调色板之上。Alfa Slab One——一种压缩的 slab 衬线体——以超大尺寸纯大写承担主要工作量。Caveat Brush 作为手写的人性化打断出现：小写，微旋转，情感温暖。DM Mono 以紧凑的大写间距承载所有元数据。调色板是电光钴蓝、琥珀橙和热红色——红色仅作为阴影/深度颜色使用，从不作为表面填充。每张幻灯片获得一层纸张颗粒叠加，使整个演示文稿具有丝网印刷感。美学响亮、自信、大众化——属于抗议标语、工会通讯或竞选巴士上的视觉语言。

People's Platform 是一套 WPA 海报遇上政治竞选的幻灯片系统——信念的视觉语言，公共演讲的图形层次。画布是温暖纸张（{colors.paper}——#F5F2EA），比白色略暖，每张幻灯片通过 CSS 伪元素获得颗粒纹理，使整个演示文稿感觉像丝网印刷而非渲染。

## 预览要素

- 调色板：blue #2C2CDC；blue-deep #1B1BB0；orange #F2A03A；orange-deep #E89321；red #E83A2A；red-deep #B7281C；cream #F4E9D6；paper #F5F2EA
- 字体排版：Alfa Slab One；Caveat Brush；Archivo Narrow；DM Mono
- 标志性手法：纸张画布（{colors.paper}——#F5F2EA）配颗粒叠加层，通过 CSS 伪元素应用于每张幻灯片。
- 标志性手法：堆叠双层 text-shadow：Npx Npx 0 {colors.red}，2Npx 2Npx 0 {colors.red-deep}。阴影大小随字体大小缩放。
- 标志性手法：Alfa Slab One 用于所有展示和结构字体——大写，行高 0.82-1.04，字间距 0.005em。
- 标志性手法：红色（{colors.red}）仅作为阴影颜色使用——从不作为表面背景。
- 标志性手法：粗重的 6px 墨色边框用于所有主要结构分割（顶栏、章节标题、列分隔符、内嵌框架）。
