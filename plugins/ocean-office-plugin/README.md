# ocean-office-plugin

演示文稿相关的 Claude Code 技能插件。

## Skills

### external-html-ppt-skill

HTML PPT 工作室 — 制作专业静态 HTML 演示文稿，支持 36 主题、31 布局、47 动画和演示者模式。

- **开源项目**：[html-ppt-skill](https://github.com/lewislulu/html-ppt-skill)

### external-frontend-slides

零依赖 HTML 演示文稿生成器，支持 34 设计模板、PPT 转换和动画效果。

- **开源项目**：[frontend-slides](https://github.com/zarazhangrui/frontend-slides)
