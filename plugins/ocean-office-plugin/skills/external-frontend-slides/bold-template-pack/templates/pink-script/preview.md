## 视觉概览

一套夜行高级定制编辑系统，以亮紫红色墨水在暗暖黑色纸张上呈现，配以柔和胶片颗粒叠加层和极细内部框架。DM Serif Display 以最大 600px 的尺寸承载每个脚本和编辑时刻；Inter 字重 300 承载正文声音；JetBrains Mono 承载精品目录元数据。美学借鉴自光面时尚杂志展开、深夜香水广告和高端杂志的编辑页面——更接近 Maison 的季度造型册而非创业演示。

Pink Script 是一套夜行高级定制编辑系统，建立在单一氛围理念之上：深暖黑色表面由左上方略暖的 #1A1218 椭圆照亮，向右下方渐变为近黑色。偏心光源读起来像是影棚柔光箱捕捉杂志展开的一角。在此之上，8% 透明度的微妙胶片颗粒叠加层使用 screen 混合模式，增添了深夜编辑摄影的颗粒感。在这个表面内部，有一个 1px 极细内部框架（纸粉色，14% 透明度，距每边内缩 36px）——每页的编辑边框。没有照明渐变、胶片颗粒和内部框架，系统会坍塌...

## 预览要素

- 调色板：ink-deep #060507；ink-violet #0F0D11；paper-blush #F5EDF1；pink #ED3D8C；pink-light #FF66A8；pink-deep #B81D67
- 字体排版：DM Serif Display；Inter；JetBrains Mono
- 标志性手法：深暖黑色表面（{components.slide-surface}）由左上方径向渐变椭圆照亮。
- 标志性手法：每张幻灯片上有微妙的胶片颗粒叠加层（{components.film-grain}）——透明度 0.08，screen 混合。
- 标志性手法：1px 纸粉色内部框架（{components.hairline-frame}）距每边内缩 36px，出现在每张幻灯片上。
- 标志性手法：亮紫红色（{colors.pink}）是唯一的彩色点缀——用作脚本颜色、导语颜色、线条颜色、药丸轮廓、行内强调和英雄脚本背后的柔和光环。
- 标志性手法：DM Serif Display 承载每个编辑时刻，尺寸从 32px 到 600px。没有第二种展示字体。
