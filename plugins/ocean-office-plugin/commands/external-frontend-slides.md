# /ocean-office:external-frontend-slides

## 何时使用此命令

当用户需求属于以下场景时，选择此命令：

- **PPT 转换**：将 .pptx 文件转换为网页演示文稿
- **创意设计**：追求独特、个性化的视觉风格，拒绝千篇一律
- **风格探索**：不确定设计偏好，希望通过视觉预览对比发现喜好
- **大胆模板**：想要非常规、有视觉冲击力的设计（赛博朋克、复古、 editorial 等）
- **增强改进**：对已有 HTML 演示文稿进行视觉升级
- **内容密度定制**：需要区分演讲型（低密度）和阅读型（高密度）设计

## 实施步骤

1. **理解需求**：询问演示文稿的用途、受众、内容准备情况和密度偏好
2. **委派执行**：调用 `external-frontend-slides` 技能完成所有制作工作
3. **交付文件**：输出可直接在浏览器打开的 HTML 文件
