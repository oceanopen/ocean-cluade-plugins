## 视觉概览

一套新粗野主义演示系统，3px 实线黑色边框即是布局。展示字体使用原生系统无衬线体栈（Segoe UI / system-ui）字重 900 严格大写——不加载任何 Web 字体。调色板是白色画布 + 黑色结构 + 两种柔和点缀色（腮红粉 #F2D4CF 和鼠尾草绿 #E5EDD6）+ 中性灰色。深度来自实黑色的硬偏移阴影，4px 和 6px——从不模糊，从不着色。美学借鉴自粗野主义编辑网页设计和杂志布局：边框无缝交汇，对比度高但被柔和点缀色温暖化，大号数字以极低透明度作为装饰性壁纸位于内容后方。整体效果锐利、系统原生、明确数字感——更接近 Notion 遇上抗议海报而非精致的提案演示。

Raw Grid 是一套新粗野主义演示系统，建立在单一结构理念之上：3px 实线黑色边框即是布局。区域之间没有边距，单元格之间没有间隙，没有圆角，没有渐变。当两个区域相遇时，一条 3px 黑线分隔它们——这条线就是整个网格系统。

## 预览要素

- 调色板：black #0A0A0A；white #FFFFFF；pink #F2D4CF；green #E5EDD6；gray #F5F5F5；darkgray #333333
- 字体排版：Segoe UI
- 标志性手法：白色（{colors.white}）画布配 3px 实线黑色（{colors.black}）边框分隔每个区域。单元格之间无间隙——边框边缘对齐。
- 标志性手法：系统无衬线体字重 900 大写用于所有展示字体；字重 500 句首大写用于所有正文字体。
- 标志性手法：实黑色的硬偏移阴影，6px 和 4px，从不模糊。
- 标志性手法：两种柔和点缀色表面——腮红粉和鼠尾草绿——作为区域填充使用，从不作为文字颜色。
- 标志性手法：标志性的黑色药丸标签组件（{components.label}）——白色大写文字在小型黑色矩形中——作为通用章节标签出现。

- 在固定舞台模型内精确构建一张 1920x1080 的标题幻灯片。
- 保留上述调色板、字体角色、表面节奏和装饰词汇。

- 永远不要在幻灯片上放置用户需求说明（如期望的氛围、受众或内部使用标签），除非用户明确希望在演示文稿中使用这些确切文字。
